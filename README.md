# Diabetes Evaluation Project

This repository demonstrates a sample medical data annotation and evaluation workflow.

## Purpose
The project simulates how medical cases can be evaluated for AI training datasets.

## Contents
- Example diabetes patient cases
- Annotation guideline
- Sample evaluation outputs

## Skills Demonstrated
- Data annotation
- Medical evaluation logic
- Dataset organization
- GitHub documentation

## Author
Medical professional transitioning into AI data evaluation.

## Annotation Workflow

1. Patient data reviewed from dataset
2. Diagnostic thresholds applied using guideline
3. Labels assigned according to medical criteria
4. Results exported as structured JSON annotations

This project demonstrates human-in-the-loop medical data annotation.
